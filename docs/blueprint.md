# **App Name**: BloodSync

## Core Features:

- User Authentication: Secure user registration and login for donors and patients.
- Geolocation Search: Find nearby blood donors using the browser's Geolocation API.
- Blood Request/Donation: Allow users to request blood or offer to donate, with specific blood type indicated.
- Smart Matching Tool: Suggests a short list of nearby users with compatible blood types in critical situations. The LLM acts as a tool, determining whether to include or exclude users from a proposed blood donation match based on up-to-date donor profiles.
- Automatic Notifications: Send email/SMS notifications for blood requests and donation matches.
- Admin Dashboard: Dashboard to manage users, blood requests, and donations.

## Style Guidelines:

- Primary color: Deep red (#B71C1C) to symbolize blood and urgency.
- Background color: Soft grey (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#008080) for call-to-action buttons and important information.
- Body and headline font: 'Inter', a grotesque sans-serif for a modern, machined, neutral look.
- Use clean, recognizable icons related to blood donation and medical symbols.
- Modern, responsive design with a clear information hierarchy.
- Subtle animations and transitions to guide users and provide feedback.
